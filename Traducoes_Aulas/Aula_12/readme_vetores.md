# Exercícios de Vetores em JavaScript

Este conjunto de arquivos contém a tradução de exercícios de vetores do VisualG para JavaScript,
utilizando Node.js e a biblioteca `readline-sync`. Cada exercício foi comentado para facilitar o estudo
da lógica e a comparação entre VisualG e JavaScript.

## Requisitos

Antes de executar qualquer arquivo, instale a biblioteca:

```bash
npm install readline-sync
```

Para executar um exercício:

```bash
node nome_do_arquivo.js
```

## Lista de Exercícios

### ex1_vetor_simples.js
Lê 5 números inteiros e exibe todos na ordem em que foram digitados.

### ex2_soma_vetor.js
Lê 10 números inteiros, soma todos e exibe o resultado total.

### ex3_contar_pares.js
Lê 8 números inteiros e conta quantos são pares.

### ex4_inverter_vetor.js
Lê 6 números inteiros e depois exibe na ordem inversa.

### ex5_buscar_elemento.js
Lê 10 números inteiros e procura um valor específico informado pelo usuário, indicando a posição se encontrado.

### ex6_acima_media.js
Lê as notas de 10 alunos, calcula a média da turma e exibe quais notas ficaram acima da média.

### ex7_vetor_para_maior_menor.js
Lê 10 números inteiros e determina o maior e o menor valor do conjunto.

### ex8_medias_p1_p2.js
Lê as notas P1 e P2 de 5 alunos, calcula a média de cada um e a média geral da turma.

### ex9_vetor_churrasco_dsm.js
Lê 5 itens de um churrasco (nome, valor unitário e quantidade), calcula o valor total de cada item e o total geral do churrasco.
Também exibe, opcionalmente, o valor aproximado por pessoa.

## Observação

Os códigos preservam a lógica original dos algoritmos em VisualG, apenas adaptando a sintaxe para JavaScript
e ajustando índices de vetor (0 a N-1). Os comentários em cada arquivo explicam passo a passo o que está sendo feito.
