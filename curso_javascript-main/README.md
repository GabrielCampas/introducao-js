Curso completo de fundamentos do JavaScript. Essas aulas foram pensadas para você que acabou de estudar [lógica de programação e algoritmos com Visualg](https://www.youtube.com/playlist?list=PLHVpcBDJr5dnK52SZvuA-cAA3DnqRJ6nz) e neste momento está tendo seu primeiro contato com uma linguagem de programação. Assim, os conceitos mais básicos de lógica de programação foram [abordados com Visualg](https://github.com/tiagotas/VISUALG_Algoritmos_Exemplo) e neste curso abordamos como "traduzir" o código para a linguagem JavaScript. Ao longo das aulas também estudaremos recursos que estão presentes apenas no JavaScript e você pode ver abaixo a relação de todas as videoaulas e o timestamp com os assuntos tratados em cada uma.

## Como ver funcionando:
- Siga a **Aula 1** e instale o ambiente de desenvolviento no seu PC
- Clone este repositório no seu PC
- Rode `npm install` na pasta do projeto (para recuperar as dependencias
- use o node para executar os arquivos javaScript, exemplo: `node app.js`
